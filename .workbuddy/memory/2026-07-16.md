# 2026-07-16 工作日志

## 修复 Vercel 部署中文乱码问题

**问题**: Godot Web 游戏部署到 Vercel 后，AI 代理返回的中文出现乱码，本地运行正常。

**根因**: `export/web/api/ai-proxy.js` 的 `readBody()` 函数使用 `body += chunk` 拼接请求体。当 chunk 是 Buffer 时，`string += buffer` 会逐个 chunk 独立做 utf-8 解码，中文字符（3字节）如果跨 chunk 边界就会被截断成乱码。本地 localhost 传输快通常单 chunk 送达不触发，Vercel serverless 经内部代理转发会拆分多个 chunk 导致必现。

**修复**: 将 `body += chunk` 改为 `chunks.push(chunk)` + `Buffer.concat(chunks).toString("utf-8")`，确保所有字节收集完再一次性解码。同步修复了 `server.js` 中相同的 `for await (const chunk of req) bodyStr += chunk` 问题。

**修改文件**:
- `export/web/api/ai-proxy.js` — readBody 函数
- `export/web/server.js` — handleAiProxy 中的 body 读取
